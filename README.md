Transcript ingestion → AI extraction of objections, blockers, buying signals, and close probability → DB persistence. Features PII-redacted viewer, semantic vector search across call history, objection drill-down with root-cause analysis, and timeline dashboards.

---

### 🏗️ Internal ERP Platform
> Next.js · PostgreSQL · Google Apps Script · Zapier

Production ERP for 300+ client accounts with KPI dashboards, rep leaderboards, RBAC, and a client portal. Batched Apps Script reporting pipeline processes the full client fleet in 30 minutes.

---

### 🗺️ Location Intelligence Tool
> JavaScript · OpenStreetMap API · Google Maps

Polygon/circle map-drawing tool with recursive area-splitting algorithm. Calls OSM API to extract zip codes, counties, towns, and cities. Deduplication + non-linear path distance filtering to reject false-radius results.

---

### 🔄 Client Onboarding Automation Pipeline
> Zapier · GoHighLevel · Gmail · Notion · Slack · JotForm

End-to-end post-contract automation triggered on proposal acceptance — welcome messaging, form delivery, CRM routing, Slack provisioning, and multi-stage reminders across 8 systems with zero manual intervention.

---

### 🕷️ Facebook Group Scraper & AI Community Predictor
> Python · AWS · OpenAI · Chrome Extension

Cookie-based scraper with human-behaviour simulation. Chrome extension with keyword/member-count filters. ML classifier using semantic text encoding + cosine similarity for community group prediction. AWS-hosted with Slack integration.
